# Сравнение алгоритмов сканирования спектральных портретов

Данный репозиторий содержит программный комплекс на языке MATLAB, предназначенный для анализа и численного сравнения двух методов сканирования спектральных портретов матричных функций: метода с фиксированным шагом и оптимизированного адаптивного метода с использованием следа (trace) матрицы.

## Обзор методов

В проекте реализованы и сравниваются:
1. **Метод М (Mal111):** Базовый алгоритм сканирования с фиксированным шагом ($h=0.01$). Он выполняет последовательный расчет нормы без учета динамики изменения значений.
2. **Новый метод (scan_with_trace_v777):** Оптимизированный алгоритм, использующий адаптивный шаг. Основные особенности:
    * Контроль изменения количества собственных значений через след (Trace).
    * Автоматическое уменьшение шага при резких изменениях нормы или индекса следа.
    * Ускорение сканирования на пологих участках.

## Структура файлов

* **`tim_c_malshv.m`**: Главный скрипт для проведения эксперимента. Он запускает оба алгоритма для матриц различной размерности ($n$), вычисляет среднее время выполнения и строит графики сравнения производительности.
* **`scan_with_trace_v777.m`**: Реализация адаптивного алгоритма сканирования с контролем следа и нормы.
* **`Mal111.m`**: Реализация алгоритма сканирования с фиксированным шагом.
* **`mat_x.m`**: Функция генерации случайных матричных функций и их сохранения в файл `mydata.mat`.
* **`mat.m`**: Вспомогательная функция для загрузки и вычисления значений матриц в конкретных точках.
